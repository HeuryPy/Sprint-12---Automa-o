README - Projeto Bootcamp de Análise de Dados
Análise de Vídeos em Alta no YouTube

====================================================
📁 Arquivos incluídos neste pacote ZIP:
----------------------------------------------------
1) trend_by_time.csv  
   → Arquivo de dados utilizado para a construção do dashboard no Tableau Public.  
   → Contém a tabela agregada com as seguintes colunas:  
      - record_id (chave primária)  
      - region (país/região)  
      - trending_date (data da tendência)  
      - category_title (categoria do vídeo)  
      - videos_count (quantidade de vídeos em alta)

2) apresentacao_projeto.pdf  
   → Apresentação em PDF contendo:  
      - Objetivo do projeto  
      - Respostas às perguntas de negócio dos gerentes  
      - Gráficos gerados no Tableau Public

3) Link para o dashboard publicado no Tableau Public:  
   https://public.tableau.com/app/profile/heury.neves/viz/sprint12_17454451079200/Painel1?publish=yes

====================================================
📌 Instruções para iniciar os arquivos:
----------------------------------------------------
1. Abra o arquivo "apresentacao_projeto_sprint12.pdf" para visualizar o relatório com as análises e conclusões do projeto.

2. Para acessar o dashboard interativo:
   - Copie e cole no navegador o link informado acima.
   - O dashboard está publicado no Tableau Public e acessível a qualquer pessoa com o link.

3. Caso deseje verificar ou utilizar os dados originais:
   - Abra o arquivo "trend_by_time.csv" em qualquer editor de planilhas (como Excel) ou ferramenta de análise de dados.
   - Esse arquivo foi utilizado como fonte de dados no Tableau.
